This project contains two files:
tagger.py: It containes the class where Tagger is implemented.
test.py: It contains the implementaion of the Tagger class. It basically imports the Tagger class and uses its method to get the desired output. 

Instructions for running the progam:

1. Open the terminal in the directory of this exercise file.
2. To run the file enter the command:
   python test.py 

The program takes doesn't take any command line arguments.

 The output contains the following details:
  - The sentence for which we need the POS tags.
  - A list of tags obtained from the Tagger for the input sentence.

  The Output.docx(or txt) files contain the output of the program run for two sentences.
